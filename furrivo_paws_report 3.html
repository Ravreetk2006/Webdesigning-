Abstract

Furrivo Paws is a fully responsive, aesthetically pleasing pet shop website built using HTML5, CSS3, JavaScript, and jQuery. The site showcases pet products, adoption listings, grooming services, and a contact form. It features a dynamic product carousel, interactive hover effects, modal pop-ups, and a mobile-friendly navigation menu. The project demonstrates modern front-end development practices including DOM manipulation, event handling, and responsive design. This report documents the design, code implementation, testing, and future scope of Furrivo Paws.
3

Introduction

In the growing pet care industry, a visually appealing and functional website is essential for any pet shop to attract customers. Furrivo Paws was conceptualized to provide an online presence that reflects warmth, trust, and professionalism. The name Furrivo combines fur and vivo (meaning life), signifying a lively space for pets.

The website includes a hero section with call-to-action, a product gallery with add-to-cart simulation, service cards, a testimonials slider using jQuery, a contact form with validation, and a responsive navbar with a hamburger menu. This report explains every component with full source code, ensuring replicability.

script.js — jQuery Interactions 
$(document).ready(function() {

  // Hamburger menu toggle
  $('.burger').click(function() {
    $('.nav-links').toggleClass('nav-active');
  });

  // Smooth scroll with header offset
  $('nav a').on('click', function(e) {
    e.preventDefault();
    $('html, body').animate({
      scrollTop: $(this.hash).offset().top - 70
    }, 800);
  });

  // Testimonial slider with auto-play
  let currentSlide = 0;
  const slides = $('.slide');

  function showSlide(index) {
    slides.removeClass('active');
    $(slides[index]).addClass('active');
  }

  setInterval(() => {
    currentSlide = (currentSlide + 1) % slides.length;
    showSlide(currentSlide);
  }, 5000);

  // Form validation
  $('#contactForm').submit(function(e) {
    e.preventDefault();
    // ... validation logic
  });
});

Challenges & Solutions

1

Responsive navbar with jQuery toggle

→ Used toggleClass and CSS transform: translateX to create a slide-in mobile menu without JavaScript positioning.

2

Autoplay slider interfering with manual navigation

→ Used setInterval with an independent index variable, allowing manual prev/next clicks to update the same state without conflict.

3

Form not clearing after successful submission

→ Called $('#contactForm')[0].reset() on the native DOM element after showing the success message.

4

Smooth scroll offset behind sticky header

→ Subtracted 70px from the scrollTop target in the jQuery animate call to compensate for header height.
